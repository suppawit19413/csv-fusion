# 🚀 CSV Fusion | Kaggle Data Merger

![CSV Fusion Mockup](https://raw.githubusercontent.com/suppawit19413/csv-fusion/main/preview.png)

A premium CSV merger tool designed specifically for Kaggle enthusiasts. Scale your data efficiently with a sleek interface and high-performance client-side processing.

## ✨ Features
- **Premium Design**: Modern glassmorphism interface with dark mode.
- **Drag & Drop**: Easily add multiple CSV files for merging.
- **Client-side Processing**: 100% browser-based. Your data never leaves your machine.
- **Smart Merge**: Automated row stacking with data preview.
- **High Performance**: Powered by PapaParse for handling large datasets.

## 🛠️ How to Use
1. Open `index.html` in your browser.
2. Drag and drop your CSV files onto the upload zone.
3. Review your files in the list.
4. Click **"Merge Files"**.
5. Preview the result and click **"Download Merged CSV"**.

## 📂 Project Structure
- `index.html`: Main application structure.
- `style.css`: Premium styling and animations.
- `app.js`: Core logic for CSV parsing and merging.
